# Icon  

The Icon property of the RadialMenu is used to customize the icon displayed in the center of RadialMenu circle.

 	 	 	                Stretch="Uniform"/>

                </Grid>

            </navigation:SfRadialMenu.Icon>



 </navigation:SfRadialMenu>
